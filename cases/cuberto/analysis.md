# Cuberto — 数字产品代理商官网

- **URL**: https://cuberto.com/
- **收录日期**: 2026-07-22
- **一句话印象**: 自定义光标的行业原型:一颗点跟了十年,hover 变形、进文字、吞媒体

## 整体气质从哪来

- Agency 站的"作品即简历",但常驻表演者只有一个:光标——克制的页面 + 戏全在指针上
- 光标不是装饰,是导览员:能点的地方它放大,有预览的地方它变成窗口
- 被全行业复刻十年,cursor 类效果的事实标准出处

## 效果清单

| # | 效果描述 | 分类 | 推测技术 | 值得抽成 pattern? |
|---|---|---|---|---|
| 1 | 光标状态机(隐藏/指针/文字/媒体四态) | cursor | lerp 跟随 + 状态类切换 | ✅ |
| 2 | 光标媒体预览(作品图/视频装进光标盒) | cursor | `-media-box` 容器随态填充 | ✅(并入 #1 变体) |
| 3 | 反色光标(blend difference,经典版) | cursor | mix-blend-mode | ✅(并入 #1 变体;现版实测 blend normal,经典手法保留在 pattern 里) |
| 4 | hover 文字态(光标里出现"查看"类动词) | cursor | `-text` 子层 + 事件委托 | ✅(同 #1) |

## 技术栈侦察

- 自研 bundle.js,无框架/动画库全局暴露(THREE/gsap/Lenis 全空)
- 光标即品牌资产:整套 `.cb-cursor` 命名体系,值得学命名不值得学抄

## 借鉴笔记

光标是用户注视时间最长的元素,把它变成状态机 = 一个元素承载全站的交互语言。四态(点/放大/文字/媒体)就是 cursor 效果的完整词汇表,再多是杂技。

## 运行时实测(浏览器注入探测,2026-07-22)

- `.cb-cursor` 五件套:`-inner` / `-text` / `-media` / `-media-box`,一个 fixed + pointer-events:none 的层
- 状态机活的:hover `<a>` 后类由 `-hidden` → `-pointer`;隐藏态泊位在屏外 `translate(-1280,-720)`(不是 display:none——保过渡)
- 现版光标 `mix-blend-mode: normal`(反色是它的经典版手法,现版改为纯色点)
- 6 个 @keyframes,页面本体动效极少——戏确实全在光标上
