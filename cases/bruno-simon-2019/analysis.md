# Bruno Simon 2019 作品集(开车看简历)

- **URL**: https://2019.bruno-simon.com/
- **收录日期**: 2026-07-15
- **一句话印象**: 史上最出圈的作品集:开一辆物理引擎小车撞遍整个简历

## 整体气质从哪来

- 把"浏览"变成"游玩":内容散落在 3D 场景里,开车去发现
- 物理引擎带来涌现乐趣(撞保龄球、拉炸药箱)
- 低多边形 + 柔和配色,性能与美术平衡

## 效果清单

| # | 效果描述 | 分类 | 推测技术 | 值得抽成 pattern? |
|---|---|---|---|---|
| 1 | 3D 物理小车探索式导航 | 3d-webgl | Three.js + Cannon.js | ☐(整体太大) |
| 2 | 物理交互物件(可撞可推) | 3d-webgl | 刚体物理 + 声音反馈 | ☐ |
| 3 | 加载序章(带进度和入场引导) | loading | 资源预载 + 动画序列 | ✅ |
| 4 | 阴影烘焙 + 低模美术风 | 3d-webgl | baked shadow 贴图 | ✅(方法论) |

## 技术栈侦察

- **完全开源**: https://github.com/brunosimon/folio-2019 ✅(Three.js + Cannon.js + Webpack,可直接跑)
- 作者即 Three.js Journey 课程作者,站内也在导流
- Awwwards 年度站点级别的影响力

## 借鉴笔记

本库最佳学习起点:唯一整站开源的顶级案例。重点读它的资源加载管线和物理-渲染同步,而不是照抄"开车"创意。
