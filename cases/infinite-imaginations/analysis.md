# infinite imaginations — Niño Ross Rodriguez 作品集

- **URL**: https://oninross.github.io/infinite-imaginations/
- **收录日期**: 2026-07-15
- **一句话印象**: 2016-2019 年代获奖作品集的活化石:hash 路由 SPA + TweenMax 手工动画

## 整体气质从哪来

- 深色底 + 打字机式文案("// about" 注释风标题)
- 菜单项带两字母缩写前缀(Ho/Ab/Ac/Co/De/Ct),程序员审美
- 全部手写框架,无重型依赖

## 效果清单

| # | 效果描述 | 分类 | 推测技术 | 值得抽成 pattern? |
|---|---|---|---|---|
| 1 | hash 路由页面过渡 | transitions | 自研 Firestarter 框架 | ☐ |
| 2 | 技能条动画 | scroll | TweenMax(GSAP 2 时代) | ☐ |
| 3 | 注释风 "// section" 标题排版 | layout | 纯排版手法 | ✅ |

## 技术栈侦察

- 框架: 自研 [Firestarter](https://github.com/oninross/firestarter)
- 动画: jQuery + TweenMax 时代产物
- **开源**: 托管在 GitHub Pages → https://github.com/oninross/infinite-imaginations ✅

## 借鉴笔记

价值在"考古":对比它和 2025 年的站能看清十年动效技术演进(TweenMax→GSAP3/anime v4,hash SPA→View Transitions)。排版手法(注释风标题)今天依然好用。
